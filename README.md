# service-request-feedback-system
Service Request &amp; Feedback Management System. Many businesses struggle with manual handling of customer complaints and feedback, causing delays and poor insights. This project automates case logging, assignment, and feedback collection, with dashboards for service performance.
This Salesforce Capstone Project focuses on solving a common business problem: manual handling of customer service requests and feedback. Businesses often face delays, missed follow-ups, and lack real-time insights into customer satisfaction and agent performance.

This project provides a Salesforce CRM solution that automates service request logging, case assignment, resolution tracking, and feedback collection, while offering dashboards for managers to monitor efficiency and satisfaction.

✅ Problem Statement

A business receives many customer complaints and service requests via calls, emails, and social channels. Currently, requests are handled manually, causing delays, missed updates, and poor insights into customer satisfaction.

The goal is to build a Salesforce-based system that:

Automates case creation and assignment.

Tracks service history for every customer.

Collects structured customer feedback.

Provides dashboards for performance analysis.

🧩 Key Use Cases

Service Request Logging – Customers submit issues via forms, automatically creating Cases in Salesforce.

Case Assignment – Cases assigned to agents based on workload or expertise using rules/flows.

Feedback Collection – After closure, customers provide feedback stored in a custom object.

Service History Tracking – Full history linked to each customer record.

Reporting & Dashboards – Manager dashboards for agent productivity and customer satisfaction.

🛠️ Technology Stack

Salesforce CRM (Admin + Developer)

Standard Objects: Case, Contact, Account

Custom Object: Feedback__c

Flows, Assignment Rules, Validation Rules

Email Alerts, Reports & Dashboards

Apex (optional for escalation logic)

Lightning App for demo presentation

🎯 Outcome

Faster and automated service request management.

Structured customer feedback collection.
